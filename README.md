Sales and Profitability Analysis 

The project focuses on identifying areas of loss and analyzing sales trends in a retail chain.

## Prerequisites
To open and interact with the analytical report, you need to have:
**Microsoft Power BI Desktop** (available for free from the Microsoft Store).

1. Power BI Report (Visualization)
An interactive dashboard providing a comprehensive view of sales performance and financial metrics.

# 1. Main KPIs and Product Ranking
This view presents the primary financial indicators (Revenue, Profit, Margin) along with a detailed ranking of products by revenue. It also includes a category structure tree highlighting areas of focus.
